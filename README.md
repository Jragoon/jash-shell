# What is jash?

jash is an interactive shell made for Unix!

# Want to try?
First, clone the repository and build the project:

``` 
$ git clone https://github.com/Jragoon/jash-shell/
$ cd jash-shell/src
$ make; make clean
```    
Then, to start the shell:

``` 
$ ./jash 
```
